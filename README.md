# Cloud-Based-Medical-Data-Management
A secure cloud-based medical management system with role-based access for Admins, Nurses, and Doctors. Admins manage users and allot patients. Nurses record patient vitals, which are encrypted via Fernet (AES &amp; SHA-256) before cloud storage Doctors can then decrypt and view allotted patients' reports. geo-login for decryption access.future notify
